# Screenshots

Add screenshots here after running the application locally.

## Recommended screenshots

1. `dashboard.png` — Portfolio overview with KPI cards and charts
2. `applications.png` — Applications table with risk score bars
3. `application_detail.png` — Side drawer with XGBoost feature importances
4. `repayment_tracker.png` — Colour-coded EMI schedule table
5. `swagger_ui.png` — FastAPI /docs Swagger interface (great for GitHub README)
6. `webhooks.png` — Webhook event log

## How to take good screenshots

- Use a 1440px wide browser window for consistency
- Populate with some sample data first (use the New Application button)
- For the Swagger UI screenshot, expand a few endpoints to show request/response schemas
